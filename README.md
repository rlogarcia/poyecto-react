## Installation

````sh
npm install
````

## Usage

To run `webpack-dev-server` for local environment:

````sh
npm run bundles

npm start
````
You can view the site at [`http://localhost:8080`](http://localhost:8080).

To bundle for deployment:

````sh
npm run bundle
````

You can view the live site at [suddi.io](https://suddi.io)
